# Curriculum Vitae Interactivo - José Azócar

Este repositorio contiene el código fuente del currículum profesional de **José Azócar**, accesible en [curriculum.joseazocar.pro](https://curriculum.joseazocar.pro). Esta plataforma ha sido diseñada como una versión digital, interactiva y optimizada del perfil profesional de José, permitiendo a reclutadores y aliados estratégicos explorar su trayectoria técnica de manera eficiente.

El sitio centraliza la experiencia laboral, formación académica y certificaciones técnicas en un entorno web de alto rendimiento y fácil lectura.

## 🚀 Características del Proyecto

* **Lectura Optimizada**: Diseño enfocado en la legibilidad (UX/UI) para facilitar la revisión de la trayectoria profesional y habilidades clave.
* **Secciones Detalladas**: Organización modular de Experiencia Profesional, Educación, Certificaciones y Habilidades Técnicas.
* **Descarga de Documentación**: Integración para la visualización o descarga del CV en formato PDF estándar para procesos de selección.
* **Diseño Totalmente Responsivo**: Adaptación perfecta a cualquier dispositivo, garantizando una presentación profesional en móviles y escritorio.
* **Rendimiento y SEO**: Optimización de etiquetas meta y tiempos de carga rápidos para asegurar una presencia digital sólida.
* **Navegación Intuitiva**: Menú de acceso rápido a las diferentes áreas de experticia y contacto directo.

## 🛠️ Tecnologías y Metodologías

Para este proyecto se han aplicado estándares modernos de desarrollo web:

* **Frontend**: HTML5 Semántico para una mejor accesibilidad y SEO, CSS3 (Flexbox/Grid) y JavaScript (ES6+).
* **Estilización**: 
    * Uso de la metodología **BEM** para mantener un código CSS limpio y escalable.
    * Librerías de iconos como **Font Awesome** y tipografía profesional vía **Google Fonts**.
* **Control de Versiones**: Git para la gestión del código y despliegue.
* **Infraestructura**: Despliegue en entorno seguro mediante HTTPS con certificados SSL/TLS.

## ⚙️ Instalación y Configuración Local

Si deseas revisar la estructura del código o realizar pruebas en un entorno local:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/azocarone/curriculum.git
    ```
2.  **Acceder al directorio:**
    ```bash
    cd curriculum
    ```
3.  **Ejecución:**
    Abre el archivo `index.html` en tu navegador de preferencia o utiliza el plugin **Live Server** en VS Code para visualizar los cambios instantáneamente.

## 📖 Uso y Navegación

* **Trayectoria**: Explora cronológicamente la evolución profesional en las áreas de desarrollo y tecnología.
* **Hard Skills**: Visualiza el dominio de herramientas técnicas, lenguajes de programación y frameworks.
* **Contacto**: Enlaces directos a perfiles profesionales (LinkedIn, GitHub) y canales de comunicación inmediata.
